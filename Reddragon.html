🔐 Pair Code (Environment Variable) Required for WhatsApp Auth
📱 How to get Pair Code:
1. Run the bot locally once using npm start or node index.js
2. The bot will display a pairing code or QR code in terminal
3. Enter the 8-digit pairing code in WhatsApp → Linked Devices → Link with Phone Number
4. OR use node pair.js if available in your red_xmd repo
5. Copy the generated PAIR_CODE and paste it above ⬆️
⚠️ Without a valid Pair Code, the WhatsApp bot won't authenticate!
